Relic Scout

Launch Website

Eko Wave, a company started by two pals and myself, just took the wraps off our first app — Relic Scout. Last week we launched a teaser site and couldn’t be prouder of the work that we’ve done so far.

I had a lot of fun creating the 8bit pixel art and applying a parallax effect to it using the jQuery plugin Plax. If you collect sports cards, magic cards, coins, or comics, you really should check it out.
